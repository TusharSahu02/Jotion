# Notion Clone

Run the app

open 2 different terminal and run below mentioned command

```
npm i
npm run dev
```

```
npx convex dev
```

## Tech stack used :

- Next.js 13
- Convex
- Clerk
- Edge Store
- Shadcn/ui
- Blacknote Editor
- Tailwind Css
